# API Tests — qa-internship.avito.com

Автоматизированные тесты API на Go для микросервиса объявлений.

## Требования

- Go 1.24+
- Доступ к интернету (тесты обращаются к `https://qa-internship.avito.com`)

## Установка зависимостей

```bash
go mod download
```

## Запуск тестов

```bash
go test -v
```

Запуск с таймаутом (рекомендуется):
```bash
go test -v -timeout 120s
```

Запуск конкретного теста:
```bash
go test -v -run TestTC001_CreateItem_Success
```

## Покрытие

### POST /api/1/item (Создание объявлений)
- TC-001: Успешное создание
- TC-002: Пустое имя
- TC-003: Отрицательная цена
- TC-004: Идемпотентность

- TC-104: Получение удалённого объявления
- TC-105: Типы данных в ответе

- TC-201: Успешное получение списка

### GET /api/1/statistics/{id} (Статистика)
- TC-301: Успешное получение (v1)
- TC-302: Успешное получение (v2)
- TC-303: Несуществующий ID (skip - зависает)

### DELETE /api/2/item/{id} (Удаление)
- TC-401: Успешное удаление
- TC-402: Повторное удаление (идемпотентность)

**Всего: 24 теста**

## Найденные баги

Всего найдено **7 багов**:
- 4 Major (валидация + утечка данных)
- 3 Minor (неправильные HTTP коды)

Подробности в `BUGS.md`

## Особенности реализации

- HTTP клиент с таймаутом 30 секунд (защита от зависания API)
- Генерация уникальных sellerID в диапазоне 111111-999999
- Независимые и воспроизводимые тесты
- Проверки с помощью assert
